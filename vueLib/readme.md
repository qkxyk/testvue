#### 使用vue库
每个Vue应用都是通过Vue函数创建一个新的Vue实例开始：
```vue
var vm= new Vue({

});
```  
##### 组件使用
```Vue
Vue.component("todo-item",{
    template:"<li>这是一个代办事项</li>"
});
```
组件使用步骤：  
1. 定义组件；
2. 使用组件。  
注：组件要定义在vue实例之前。  
###### 组件传递参数
1. 在组件定义是使用props,并在模板中使用参数；
2. 在组件调用时传入参数，v-bind 指令将待办项传到循环输出的每个组件中
